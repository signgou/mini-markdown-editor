# 快捷键

|   功能   | 快捷键（Mac） | 快捷键（Windows） |
| :------: | :-----------: | :---------------: |
|   加粗   |     ⌘ + B     |     Ctrl + B      |
|   斜体   |     ⌘ + I     |     Ctrl + I      |
|  下划线  |     ⌘ + U     |     Ctrl + U      |
|  删除线  | ⌘ + Shift + X | Ctrl + Shift + X  |
|   引用   | ⌘ + Shift + 9 | Ctrl + Shift + 9  |
| 无序列表 | ⌘ + Shift + 8 | Ctrl + Shift + 8  |
| 有序列表 | ⌘ + Shift + 7 | Ctrl + Shift + 7  |
| 内联代码 |    ⌘ + \`     |     Ctrl + \`     |
|  代码块  |  ⌘ + Alt + C  |  Ctrl + Alt + C   |
|   链接   |     ⌘ + K     |     Ctrl + K      |
|   表格   |  ⌘ + Alt + T  |  Ctrl + Alt + T   |
|   撤销   |     ⌘ + Z     |     Ctrl + Z      |
|   重做   | ⌘ + Shift + Z | Ctrl + Shift + Z  |
|   全屏   |  ⌘ + Alt + F  |  Ctrl + Alt + F   |
|   保存   |     ⌘ + S     |     Ctrl + S      |
