# Markdown 语法

## 标题

```md
# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题
```

## 加粗、斜体、下划线、删除线

输入：

```md
**加粗**

_斜体_

--下划线--

~~删除线~~
```

输出：

**加粗**、_斜体_、<u>下划线</u>、~~删除线~~

## 引用

输入：

```md
> 引用一段内容
```

输出：

> 引用一段内容

## 分割线

输入：

```md
---
```

输出：

---

## 列表

输入：

```md
<!-- 无序列表 -->
- 列表1
- 列表2
- 列表3

<!-- 有序列表 -->
1. 列表1
2. 列表2
3. 列表3
```

输出：

无序列表：

- 列表1
- 列表2
- 列表3

有序列表：

1. 列表1
2. 列表2
3. 列表3

## 内联代码

```md
`内联代码`
```

输出：

`内联代码`

## 代码块

输入：

<div>
\`\`\`js<br/>
console.log('代码块')<br/>
\`\`\`
</div>

输出：

```js
console.log('代码块')
```

## 链接

输入：

```md
[链接文字](链接地址)
[链接文字](https://www.example.com)
```

输出：

[链接文字](https://www.example.com)

## 图片

输入：

```md
![图片描述](图片地址)

![图片描述](https://www.example.com/image.jpg)
```

输出：

<img src="/logo.svg" width={100} />

## 表格

输入：

```md
| 表头1 | 表头2 | 表头3 |
| ------ | ------ | ------ |
| 内容1 | 内容2 | 内容3 |
```

输出：

| 表头1 | 表头2 | 表头3 |
| ------ | ------ | ------ |
| 内容1 | 内容2  | 内容3  |
